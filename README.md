# PharmaCRM — CRM для фармацевтической компании

Мобильное React Native приложение (Expo) для управления продажами, compliance, логистикой и полевой работой медицинских представителей.

## Запуск

```bash
npm install --legacy-peer-deps
npm run web
```

Откроется в браузере: **http://localhost:8081**

Если порт занят:
```bash
npx expo start --web --localhost --port 8082
```

Для телефона (Expo Go):
```bash
npm start
```

### Демо-вход

| Email | Пароль | Роль |
|-------|--------|------|
| `ivanov@pharma.ru` | любой | Медпредставитель (2FA) |
| `petrov@pharma.ru` | любой | Менеджер продаж |
| `admin@pharma.ru` | любой | Администратор |

**2FA код:** любой 6-значный код (например `123456`)

## Модули

### Compliance и аудит
- **Audit Trail** — журнал изменений с IP/устройством, архивирование, экспорт, защита критичных записей
- **Электронные подписи** — подписание документов/визитов с 2FA, блокировка после подписи, валидация
- **Согласования** — маршруты (параллельные/последовательные), дедлайны, комментарии, возврат на доработку
- **Согласия** — управление согласиями HCP, запрет коммуникаций без согласия
- **Подарки и расходы** — Transfer of Value, лимиты, проверка конфликта интересов

### Склад и GDP
- Остатки по складам и сериям, сроки годности, резервирование
- Поставки с температурным контролем, трассировка партий, recall

### Полевая работа
- Визиты с check-in/check-out и геолокацией
- Карта клиентов и маршрут на день
- Offline-режим и синхронизация

### KPI и аналитика
- Планы продаж, KPI медпредставителей и менеджеров
- ABC/XYZ анализ, дашборды, отчёты

### Документооборот
- Хранилище документов с версиями и правами доступа

### Интеграции (заглушки)
- 1С/SAP/ERP, Power BI, PV-система, email/телефония

### Безопасность
- Роли и права, 2FA, SSO-ready архитектура
- Шифрование подписей (SHA-256)

### Автоматизация
- Правила: автозадачи, напоминания, compliance-проверки
